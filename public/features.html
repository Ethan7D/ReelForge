痛点优先级矩阵

我们重点解决什么

以「角色先建」为纲，以「剧本规避」为策，以「批量筛选 + 一致性校验」为兜底。

角色一致性

P0 致命

同一角色切镜头后「变脸」：发型变、五官漂移、服装颜色不同。

应对：角色先建 + LoRA + 参考图 + 换脸 + 校验
可解决程度 ≈ 70%（初期 70% 自动 + 30% 人工兜底）

场景一致性

P0 高

同一房间换角度后布局变化，窗户位置移动、光影不一致。

应对：多角度参考图 + 场景档案 + 提示词锁定
可解决程度 ≈ 50%

口型同步

P1 中

角色嘴型与台词不匹配，中文场景尤其明显。

应对：先无声视频 → TTS → Wav2Lip 驱动口型
可解决程度 ≈ 80%

镜头拼接与节奏

P1 高

单次生成 5-15 秒，一集需 6-20 个镜头拼接，拼接处跳变。

应对：镜头编排器 + 相邻镜头一致性校验
可解决程度 ≈ 60%

动作 / 物理

P1 中高

走路滑步、手部穿模、物理接触逻辑错误。

应对：剧本阶段规避复杂动作 + 批量筛选废片
可解决程度 ≈ 30%（以规避为主）

光影连贯

P2 中

同场景不同镜头色调跳变，阴影方向不一致。

应对：光影提示词锁定 + 后期 LUT 统一调色
可解决程度 ≈ 40%（后期调色兜底）
